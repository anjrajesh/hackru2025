# SafeRoute Project

This repository contains the full-stack SafeRoute application with AI-powered incident categorization.

## Quick Start

See the main [README.md](./README.md) for detailed setup instructions.

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm start
```
